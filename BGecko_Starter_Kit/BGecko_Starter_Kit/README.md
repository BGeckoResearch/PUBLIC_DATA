# BGecko Starter Kit (v0.1)

This folder is a **world-class operating package** for BGecko as a starting organization.

**Mission (from bgecko.org):**
- Transform ideas into initiatives.
- Publish reproducible research, learning pathways, tools, and mission-backed products.

**How to use this kit**
1. Start in **01_Plans_and_Execution** and run the 30-day operating cadence.
2. Use **04_Integration_Team** to keep systems sane as you grow.
3. Use **02_Research** + **03_Institute_(Learning)** as your core "publish" engines.
4. Use **05_Studio** to make everything look/feel coherent.
5. Use **06_BGecko_Shop** to convert attention into mission-backed revenue.

**No false data policy**
All spreadsheets/templates are intentionally initialized with:
- empty rows, placeholders, and formulas;
- example entries clearly labeled as examples;
- no invented metrics, prices, or performance claims.
