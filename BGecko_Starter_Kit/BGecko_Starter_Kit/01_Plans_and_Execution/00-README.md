# Plans and Execution

**Purpose:** Turn BGecko's mission into weekly shipped outcomes.

**Cadence**
- Daily: 15 min standup (async is ok).
- Weekly: planning + review (ship list, blockers, next bets).
- Monthly: strategy review (OKRs + budget + capacity).

**Key artifacts in this folder**
- OKRs + roadmap templates
- Project charter template
- Risk register + decision log
- A lightweight Operating System PDF (roles, rituals, definitions)
