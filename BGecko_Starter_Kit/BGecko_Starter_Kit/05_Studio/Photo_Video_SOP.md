# Photo/Video SOP (v0)

## Capture
- Shoot in consistent lighting; avoid mixed color temperatures.
- Capture: wide + detail + lifestyle.
- Keep RAW originals.

## Edit
- Apply the same baseline corrections (exposure, contrast, white balance).
- Export 2x resolution for web.

## Publish
- Filenames: YYYY-MM-DD_project_asset_vN.ext
- Alt text for images.
- Captions/transcripts for video.

## Archive
- Store originals + exports + license proofs together.
