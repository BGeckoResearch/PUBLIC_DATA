# Asset Library Structure (recommended)

```
assets/
  brand/
    logos/
    colors/
    typography/
  web/
    banners/
    icons/
  merch/
    mockups/
    print_files/
  video/
    raw/
    exports/
    captions/
  audio/
    licensed/
    commissioned/
  docs/
    diagrams/
    figures/
```

**Metadata (required for every asset)**
- Source URL or creator
- License type + proof (file or screenshot)
- Intended use (web, shop, paper, video)
- Date + version
