# Lecture / Lesson Outline (template)

## Hook (1 minute)
- A concrete question or paradox.

## Setup (3-5 minutes)
- Definitions (explicit)
- One motivating example

## Core development (10-20 minutes)
- Derivation / proof sketch
- Visual intuition (diagram)
- Edge cases and failure modes

## Practice (10-20 minutes)
- 1 easy exercise
- 1 medium exercise
- 1 hard transfer exercise

## Close (1 minute)
- What this unlocks next
- Link to reading + exercises + feedback form
